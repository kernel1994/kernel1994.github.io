Matlab, Numpy, Python, 图像处理, 笔记

在学习图像处理（有空再整理出文章）的过程中，是将 Matlab 代码翻译为 Python 代码的过程，主要使用 Numpy 库，由于 Matlab 与 Numpy 之间的差异，导致一些困难，故整理出本文章来记录。

2017年6月，K-means

关于读取图片

  • 若使用 skimage.io.imread() 则图像像素值为 [0, 255] 的整数
  • 若使用 matplotlib.pyplot.imread() 则图像像素值为 [0, 1] 的浮点数， 即 x / 255

关于均值函数 mean()

  • 在 Matlab 中不指定轴(axis) 默认是 axis=0，结果为行向量。
  • 在 Numpy 中不指定轴(axis) 默认将数组扁平化后求均值，结果为一个数。

所以今后编码过程中都应将明确意图，尽量少使用默认值，提高代码可读性。

rank 1 array

类似于 (3,) 这样的 rank 1 数组在代码中尽量避免使用，需要继续用 reshape([1, 3]) 来明确维度
P.S. reshape 是 O(1) 的时间复杂度，所以放肆地调用吧
