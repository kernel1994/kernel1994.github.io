用户与组

用户管理

  • 添加用户：adduser LOGIN
  • 修改用户信息：usermod [options] LOGIN
    • 修改用户说明：usermod -c COMMENT LOGIN
    • 设置用户初始组：usermod -g GROUP LOGIN
    • 将用户[添加]进组：usermod [-a] -G GROUP1[,GROUP2,…[,GROUPN]]] LOGIN
    • 修改用户名同时修改 home 目录名：usermod -l NEW_LOGIN -d HOME_DIR -m LOGIN（-m 命令表示移动原来 home 目录的内容到新 home 目录中）

组管理

  • 新建组：groupadd GROUP
  • 将多个用户加入到组：gpasswd -M USER1[,USER2,…[,USERN]]] GROUP
