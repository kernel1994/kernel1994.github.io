lambda 表达式

语法

方法引用 (method reference)

方法引用有三种形式：

  • objectName::instanceMethod
    等价于: (arg) -> { objectName.instanceMethod(arg); }
  • ClassName::staticMethod
    等价于: (arg) -> { ClassName.staticMethod(arg); }
  • ClassName::instanceMethod
    等价于: (arg0, rest) -> { arg0.instanceMethod(rest); }

前两种把 lambda 表达式的参数直接当成 instanceMethod | staticMethod 的参数来调用。
其中第一种方式中的 objectName 为已经存在的实例变量。
例如：

  • System.out::println 等同于 x -> System.out.println(x);
  • Math::max 等同于 (x, y) -> Math.max(x, y);

第三种把 lambda 表达式的第一个参数当成 instanceMethod 的目标对象，其他剩余参数当成该方法的参数。
例如：

  • String::substring 等同于 (str, i) -> str.substring(i);

构造方法引用

ClassName::new
例如：

  • Supplier c1 = Apple::new; 等同于 Supplier c1 = () -> new Apple();
